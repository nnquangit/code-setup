# IDE Setup

### Documentation

```
code --install-extension cschlosser.doxdocgen
```

### Formatter & linter

```
code --install-extension esbenp.prettier-vscode
code --install-extension dbaeumer.vscode-eslint
code --install-extension ms-vscode.vscode-typescript-tslint-plugin
```
